# 3. Page série (nouvelle approche)

## Objectif

Créer une page qui devient le **hub central d'une série**. Elle ne doit pas seulement présenter la série, mais être l'endroit où un fan peut tout faire : découvrir, suivre, discuter et interagir.

## Contenu proposé

### Hero section immersive

- Affiche de la série.
- Bande-annonce intégrée.
- Image d'ambiance (background dynamique).
- Note globale de la communauté.
- Nombre de fans.
- Statut personnel :
  - "Je regarde"
  - "J'ai terminé"
  - "Je veux voir"
  - "Abandonnée"

### Résumé intelligent

Une présentation courte mais engageante :

- Synopsis.
- Genre.
- Nombre de saisons.
- Durée moyenne des épisodes.
- Année de diffusion.
- Plateformes disponibles.

### Progression personnelle

Pour chaque utilisateur :

```
Vous avez regardé :
Saison 3 - Episode 5 / 10

Progression :
████████░░ 80%

Prochain épisode :
S03E06 - "Titre"
```

### Zone communauté

Créer l'aspect fan :

- Discussions populaires.
- Théories des fans.
- Citations cultes.
- Moments préférés.
- Personnages favoris.
- Classement des meilleurs épisodes.

### Timeline de la série

Une vue chronologique :

```
2018
 └ Saison 1

2019
 └ Saison 2

2021
 └ Saison 3
```

### Recommandations personnalisées

"Vous aimez cette série, vous pourriez aimer :"

- Séries similaires.
- Séries regardées par vos amis.
- Séries populaires auprès de profils similaires.

# 4.Page saison seule

## Objectif

Créer une immersion dans une saison spécifique.

Exemple :
**Stranger Things - Saison 4**

## Contenu

### Header saison

- Image spécifique de la saison.
- Synopsis de la saison.
- Date de sortie.
- Nombre d'épisodes.
- Progression personnelle.

Exemple :

```
Saison 4

Vous avez vu :
6 / 9 épisodes

Dernière activité :
Episode 6 regardé hier
```

---

## Liste des épisodes

Chaque épisode devient une carte :

```
Episode 05
Titre : "The Nina Project"

Durée : 52 min

Synopsis :
Eleven découvre...

Statut :
✓ Vu

Note personnelle :
★★★★★

Commentaires :
128 discussions
```

---

## Mode découverte

Permettre de masquer les spoilers.

Options :

- Voir uniquement les épisodes non vus.
- Voir les épisodes populaires.
- Voir les épisodes les mieux notés.

# 5.Page épisode seul

Transformer un épisode en expérience communautaire.

---

## Contenu

### Présentation épisode

- Image principale.
- Titre.
- Numéro épisode.
- Durée.
- Date de diffusion.

---

### Synopsis

Deux versions :

- Sans spoiler.
- Avec spoiler (débloqué après validation).

---

Après avoir regardé :

"Que pensez-vous de cet épisode ?"

Actions :

- Noter.
- Ajouter une réaction.
- Écrire un commentaire.
- Ajouter aux favoris.

### Discussion épisode

Créer une discussion spécifique :

```
542
```

- Citation.

- Réaction.
